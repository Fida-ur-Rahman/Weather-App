# Weather App

A responsive **Weather App** built using **HTML, CSS, and JavaScript**. This application fetches real-time weather data from a public API and displays the current temperature, humidity, and weather conditions for any city entered by the user.

##  Features

- Search for weather by city
- Displays temperature, humidity, and weather description
- Real-time weather updates from an API
- Responsive and mobile-friendly design
- Interactive and user-friendly interface

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API (or any public weather API)

##  How It Works

1. Users enter a city name in the search input.
2. JavaScript fetches data from the weather API.
3. The app displays the city’s current weather, temperature, humidity, and conditions.
4. The interface updates dynamically without page reload.

##  How to Use

1. Clone the repository:
   ```
   git clone https://github.com/Fida-ur-Rahman/Weather-App.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
4. Enter a city name to see the current weather information.
